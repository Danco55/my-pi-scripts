Password Hasher Tool 🛡️

A simple Python script to hash passwords using MD5, SHA-256, and SHA-512.  
This project is perfect for learning basic Python and cryptography concepts.

1. Run the script -> password_hasher.py
2. Enter your password when prompted
3. The script outputs hashes in the terminal
4. It also saves them to -> hashed_passwords.txt

Why is it usefull?

- Learn how to generate hashes for passwords  
- Understand the difference between MD5, SHA-256, and SHA-512  
- Foundation for cryptography and cybersecurity projects
